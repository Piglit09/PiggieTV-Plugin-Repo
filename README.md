# 🚀 About

PiggieTV transforms the standard Jellyfin experience into a modern, premium streaming interface inspired by platforms like Netflix, Disney+, Apple TV, and Plex — while remaining fully self-hosted and powered by Jellyfin.

This repository provides the official PiggieTV Jellyfin plugin repository used for:
- Plugin distribution
- Updates
- Frontend integrations
- Dashboard enhancements
- Recommendation systems
- Signup systems
- UI feature deployment
